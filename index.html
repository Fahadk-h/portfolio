Projects

Spotify tracks Dashboard

Overview:

This project involved analyzing the Most Streamed Spotify Songs of 2023 dataset, published by Nidula Elgiriyewithana on Kaggle. The primary objective was to extract insights such as the highest streamed track, song characteristics, and release patterns over time.

>Key Questions Explored

  1. How can ChatGPT be leveraged to enrich the dataset and use external tools like the Spotify Developer API?.
  2. Which track has the highest number of streams?
  3. How do average streams per year compare to the top tracks?
  4. What characteristics define popular songs?
  5. What is the release pattern of these tracks?

>Process & Tools

  1. Data Enrichment with ChatGPT

    2. Initially, I attempted to enhance the dataset using ChatGPT to add song or album URLs. However, due to limitations in internet connectivity, I shifted to using the Spotify Developer API. I successfully appended the URL column by executing a custom Python script, providing direct links to the song covers.

  2. Handling Missing Data

    3. Upon reviewing the dataset, I identified the absence of a date column. Utilizing the available information (year, month, and day), I created a calculated column with the "DATE" function in Power BI. To enhance time-based analysis, I employed Bravo, an external tool, to generate a calendar table.

>Visualizations

  • A line chart illustrating the number of tracks released over time, leveraging the calendar table to implement time intelligence.
  • Slicers for Date Range, Artist, and Track, enabling users to filter data and derive additional insights.
  • Bar charts displaying stream counts for each track and artist, highlighting the highest streamed songs and popular artists.
  • KPI cards presenting key data points such as the highest streamed track, total streams, and the number of tracks per artist.
  • HTML card to display album artwork from the URL column, enhancing the visual appeal of the dashboard and providing instant context for the tracks and artists, making the experience more engaging for users.

Key Insights:
  • The track “Shape of You” by Ed Sheeran ranked second in total streams. However, Bad Bunny outperformed with 26 tracks and 15.4 billion streams, compared to Sheeran's 13 tracks and 14.6 billion streams.
  • The dashboard offers interactive insights, allowing users to explore data by track, artist, and release date.

Impact and Outcomes:

The interactive dashboard provided valuable insights into streaming trends, allowing stakeholders to gain a deeper understanding of patterns in song popularity and artist performance. By integrating external tools and APIs, the project not only enhanced the dataset’s value but also showcased proficiency in data enrichment and API utilization. This integration helped to significantly improve the usability of the data, making it a more powerful tool for analysis and decision-making.
